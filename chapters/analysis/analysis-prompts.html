22.1 Multilevel Modeling

Setting up a multilevel model in R:

I am running a multilevel model in R using lme4. My data have 
students nested within schools (N=342 students, N=18 schools). 
The outcome is sel_total_mean (continuous). Level-1 predictors 
are grade and gender. The level-2 predictor is school_size. 
Write code to: (1) run a null model and calculate the ICC, 
(2) add level-1 predictors, (3) add the level-2 predictor, 
and (4) compare models. Use broom.mixed to extract results.

interpret the fixed effects, random effects, and variance 
components. Flag anything that looks unusual or worth 
investigating further.

22.2 Meta-Analysis

Setting up a meta-analysis in R:

I am running a meta-analysis in R using metafor with robust 
variance estimation (RVE) via the clubSandwich package. I have 
effect sizes in a dataframe called `meta_data` with columns: 
study_id, effect_id, yi (effect size), vi (variance), and 
moderator variables [list them]. Write code to fit a multilevel 
random effects model with RVE, produce a forest plot, and test 
for publication bias using Egger's test.

22.3 Table and Figure Production

Producing a results table in Word:

I have model results stored in a dataframe called `results` in R 
with columns: term, estimate, std.error, conf.low, conf.high, 
p.value. Write code using flextable and officer to produce an 
APA-style results table and save it as a Word document at 
output/tables/model_results.docx. Use Times New Roman 12pt, 
include a table title, and format p-values to three decimal places.

Saving a ggplot figure:

I have a ggplot object called `p` in R. Write code to save it 
as both a PNG (300 dpi, 6x4 inches) and a PDF to 
output/figures/figure1. Use the here package for the file path.
