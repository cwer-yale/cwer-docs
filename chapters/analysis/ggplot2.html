16  Visualization with ggplot2

ggplot2 is R’s primary visualization package and one of the most widely used data visualization tools in data science. It builds plots layer by layer using a consistent grammar. Every ggplot starts with ggplot(), specifies which variables map to which visual properties using aes(), and then adds layers using geom_*() functions.

Load it as part of the tidyverse:

library(tidyverse)

16.1 Basic Plot Structure

Every ggplot has three essential components:

  1. Data — the dataframe you are plotting
  2. Aesthetics (aes()) — which variables map to which visual properties (x axis, y axis, color, size, etc.)
  3. Geometry (geom_*()) — what type of plot to draw
ggplot(data, aes(x = grade, y = sel_total_mean)) +
  geom_boxplot()

Layers are added with +. Each additional layer refines or adds to the plot.

16.2 Common Plot Types

16.2.1 Histogram

Use for visualizing the distribution of a single continuous variable:

ggplot(data, aes(x = sel_total_mean)) +
  geom_histogram(bins = 20) +
  labs(
    x = "SEL Total Mean",
    y = "Count",
    title = "Distribution of SEL Scores"
  )

16.2.2 Bar Chart

Use for comparing means or counts across groups:

data |>
  group_by(grade) |>
  summarise(mean_sel = mean(sel_total_mean, na.rm = TRUE)) |>
  ggplot(aes(x = factor(grade), y = mean_sel)) +
  geom_col() +
  labs(x = "Grade", y = "Mean SEL Score")

16.2.3 Box Plot

Use for showing distributions across groups:

ggplot(data, aes(x = factor(grade), y = sel_total_mean)) +
  geom_boxplot() +
  labs(x = "Grade", y = "SEL Total Mean")

16.2.4 Scatter Plot

Use for examining the relationship between two continuous variables:

ggplot(data, aes(x = rdg_score, y = sel_total_mean)) +
  geom_point(alpha = 0.5) +
  geom_smooth(method = "lm") +
  labs(x = "Reading Score", y = "SEL Total Mean")

The alpha argument controls point transparency (0 = invisible, 1 = solid), which helps when points overlap. geom_smooth(method = "lm") adds a linear regression line with a confidence interval.

16.2.5 Line Plot

Use for showing change over time or across waves:

data |>
  group_by(wave, grade) |>
  summarise(mean_sel = mean(sel_total_mean, na.rm = TRUE)) |>
  ggplot(aes(x = wave, y = mean_sel, color = factor(grade), group = factor(grade))) +
  geom_line() +
  geom_point() +
  labs(x = "Wave", y = "Mean SEL Score", color = "Grade")

16.3 Labels and Titles

Always label your axes and add a title. Use labs():

ggplot(data, aes(x = grade, y = sel_total_mean)) +
  geom_boxplot() +
  labs(
    title = "SEL Scores by Grade",
    subtitle = "Wave 1 data, N = 342",
    x = "Grade",
    y = "SEL Total Mean Score",
    caption = "Source: CWER Wave 1 Survey"
  )

16.4 Themes

Themes control the overall appearance of the plot. theme_minimal() is a clean, publication-friendly option:

ggplot(data, aes(x = grade, y = sel_total_mean)) +
  geom_boxplot() +
  labs(x = "Grade", y = "SEL Total Mean") +
  theme_minimal()

Other useful themes: theme_classic(), theme_bw().

16.5 Saving Figures

Save figures to output/figures/ using ggsave(). Always assign your plot to an object first:

p <- ggplot(data, aes(x = factor(grade), y = sel_total_mean)) +
  geom_boxplot() +
  labs(x = "Grade", y = "SEL Total Mean") +
  theme_minimal()

ggsave(
  here("output", "figures", "sel_by_grade.png"),
  plot = p,
  width = 6,
  height = 4,
  dpi = 300
)

Use dpi = 300 for publication-quality resolution. Standard figure widths are 6 inches for a single-column figure and 8 inches for a full-width figure.

For PDF output (preferred for manuscripts):

ggsave(
  here("output", "figures", "sel_by_grade.pdf"),
  plot = p,
  width = 6,
  height = 4
)
